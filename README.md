# MySQL Sales Analysis Project

## Overview

This project analyzes a sales dataset using MySQL to extract meaningful business insights related to customer behavior, product performance, and revenue trends.

## Objectives

* Analyze overall sales performance
* Identify top-performing products
* Understand customer purchasing patterns
* Evaluate revenue distribution

## Key SQL Techniques Used

* JOINs (INNER, LEFT)
* Aggregations (SUM, AVG, COUNT)
* GROUP BY and ORDER BY
* Subqueries

## Project Structure

* `mySQL sales analysis.sql` → Contains all queries used for analysis

## Sample Insights

* Identified highest revenue-generating products
* Analyzed customer purchase frequency
* Evaluated trends in sales performance

## How to Run

1. Open MySQL Workbench or any SQL environment
2. Import dataset (if applicable)
3. Run the SQL file
4. Review outputs
